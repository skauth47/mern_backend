Available endpoints

Use these routes for CRUD operations, bulk inserts, and account access. JSON request bodies are required for write operations.

GET /api/users List all users
POST /api/users Create a user
PUT /api/users/:id Update a user
DELETE /api/users/:id Delete a user
POST /api/users/bulk Create multiple users
POST /api/users/register Register an account
POST /api/users/login Log in to an account
